## Setup

1. cd frontend
2. npm i or yarn
3. add .env.local file in `/frontend` with key value pairs as in env.example. Take API keys from https://huddle01.com/docs/api-keys
4. add .env file with key as `DATABASE_URL` and pair as your database url.
5. npm run dev
